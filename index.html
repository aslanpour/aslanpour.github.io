About Us

Who Am I?

Hi I'm Mohammad Sadegh Aslanpour. I Researcher at CSIRO's DATA61 and PhD Student at Monash Uni.

My research interests include: Cloud/Fog/Edge/Serverless Computing, Autonomous Systems, Simulation, Optimization and Internet of Things.

What research I do?

Here are some of my expertise

Auto-scaling

Many algorithms for auto-scaling mechanisms have been developed.

Three-tier Web Application

Three-tier deployment of the Web applications in cloud platforms

Simulation

Simulating cloud environments supporting dynamic resource provisioning

Artificial Neural Networks

Workload prediction for the Web applications using MLP and RBF

OpenStack

Web application deployment using OpenStack platform

Optimization

Optimization of cloud applications in both simulation and real test-beds
